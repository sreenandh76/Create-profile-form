# Profile Creator Form

## Short Description

A responsive user registration form built using HTML, CSS, and JavaScript that performs real-time validation for name, email, and password fields. The submit button remains disabled until all inputs are valid.

## Live Demo

Here’s the live link to view the project: 

https://sreenandh76.github.io/Create-profile-form/

## Features

* Real-time form validation
* Name field cannot be empty
* Email format validation
* Password must be at least 6 characters
* Submit button disabled until all fields are valid
* Responsive design for mobile and laptop

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Purpose

This project was created to practice JavaScript DOM manipulation and client-side form validation.
